# paineis_bi_vue

### Configurar projeto (node v16.10)
```
sudo apt install curl
```
```
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
```
```
nvm install 16.10
```
```
nvm use 16.10
```
```
yarn install
```

### Executar o projeto
```
yarn serve
```

### Gerar o build para produção
```
yarn build
```
